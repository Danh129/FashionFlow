# 👕 FashionFlow - Website Kinh Doanh Thời Trang Hiện Đại

![UI/UX](https://img.shields.io/badge/Design-Responsive-brightgreen?style=for-the-badge)

## 📌 Giới thiệu dự án
FashionFlow là một nền tảng thương mại điện tử chuyên biệt cho ngành thời trang.
Website không chỉ là nơi trưng bày sản phẩm mà còn tập trung tối ưu trải nghiệm người dùng từ bước tìm kiếm, lựa chọn size cho đến khi hoàn tất đơn hàng.

## ✨ Tính năng chính
* 🛍️ **Danh mục sản phẩm thông minh:** Lọc theo loại đồ (Áo, Quần).
* 🔍 **Lọc theo giá:** Lọc sản phẩm với mệnh giá ( < 200, > 500...)
* 🛒 **Giỏ hàng:** Thêm nhanh sản phẩm vào giỏ, cập nhật số lượng.
* 📱 **Giao diện Responsive:** Hiển thị hoàn hảo với giao diện đẹp.
* 🔍 **Tìm kiếm mạnh mẽ:** Tìm kiếm sản phẩm theo tên một cách thông minh.
* 🛒 **Bảng Size:** Giúp khách hàng tìm kiếm sản phẩm phù hợp với nhu cầu.
* 🛍️ **Xóa Sản Phẩm:** Hổ trợ xóa sản phẩm khi người dùng không còn nhu cầu mua nữa.

## 🏗 Công nghệ sử dụng
* **Frontend:** [HTML5 & CSS] - Xây dựng giao diện mượt và đẹp, đầy sang trọng.
* **Backend:** [JavaScript] - Xử lý nghiệp vụ cho phần chức năng ( tìm kiếm, lọc, thêm vào giỏ hàng....)
* **Tooling:** Git, Visual Studio Code.

## 🗂 Cấu trúc dự án
* `index.html`: Trang chủ hiển thị danh sách sản phẩm.
* `css/`: Chứa các file định dạng giao diện sang trọng.
* `js/`: Xử lý logic giỏ hàng, tìm kiếm và lọc giá.

## 🚀 Hướng dẫn khởi chạy
1. Tải và cài đặt **Visual Studio Code**.
2. Mở thư mục dự án trong VS Code.
3. Cài đặt Extension **Live Server**.
4. Chuột phải vào file `index.html` và chọn **Open with Live Server** để bắt đầu trải nghiệm mua sắm!
